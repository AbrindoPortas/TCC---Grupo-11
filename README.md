## Descrição
O sistema auxilia adultos tardiamente diagnosticados com TEA com suas dificuldades no cotidiano, seja como lidar com as suas emoções, até um forum para o adulto se expressar e falar como se sente

## Como executar o projeto (passo-a-passo)
1.selecione a pasta meu-site e abra o terminal integrado
2.instale as dependencias(
npm install express express-session express-handlebars sqlite3 multer bcrypt bcryptjs

--opcional para o criaadm.js--

npm install bcryptjs

)
3.execute o codigo node app.js
